QUIC Test Page

This webpage uses an experimental Go implemention of the novel QUIC protocol proposed by Google.

To use it, you need to use a current Chrome version, and launch it using a special command from the console. For OS X, try: /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:7000,quic.clemente.io:8005,quic.clemente.io:8006,quic.clemente.io:8007,quic.clemente.io:8008,quic.clemente.io:8009 --enable-quic https://quic.clemente.io:7000 For linux, try: chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:7000,quic.clemente.io:8005,quic.clemente.io:8006,quic.clemente.io:8007,quic.clemente.io:8008,quic.clemente.io:8009 --enable-quic https://quic.clemente.io:7000
