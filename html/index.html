This webpage demoes an experimental Go implemention of the novel QUIC protocol proposed by Google.

To try it out, you need to use a current Chrome version, and launch it with a special command from the console. Unfortunately, only beta (v51) and dev (v52) support our experiments listed below right now, but you can still load this demo page using the current stable version (v50).

Stable

For OS X, try: /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:443 --enable-quic https://quic.clemente.io For linux, try: chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:443 --enable-quic https://quic.clemente.io

Beta and Dev

For OS X, try: /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:443,quic.clemente.io:8005,quic.clemente.io:8006,quic.clemente.io:8007,quic.clemente.io:8008,quic.clemente.io:8009 --enable-quic https://quic.clemente.io For linux, try: chrome --user-data-dir=/tmp/chrome --origin-to-force-quic-on=quic.clemente.io:443,quic.clemente.io:8005,quic.clemente.io:8006,quic.clemente.io:8007,quic.clemente.io:8008,quic.clemente.io:8009 --enable-quic https://quic.clemente.io
